SECTION 02

형 변환 — int(), float(), str()

형 변환(type conversion)은 한 객체의 값을 가지고 다른 타입의 새 객체를 만드는 일입니다. 1주차에서 본 객체 모델로 다시 강조합니다 — 원래 객체를 바꾸는 게 아니라, 새 객체를 만드는 것입니다.

세 가지 변환 함수

  • int(x) — x를 정수 객체로 변환
  • float(x) — x를 실수 객체로 변환
  • str(x) — x를 문자열 객체로 변환

동작 순서:

  1. 조건 검사: count < 5가 True인가?
  2. True면 블록 실행
  3. 다시 1번으로 → 조건 검사

위험: 무한 루프

조건을 절대 False로 못 만들면 무한히 반복합니다. 초보자가 자주 만드는 실수 — count = count + 1을 깜빡하는 경우입니다.

무한 루프 주의

이 강의 환경에서는 무한 루프를 만들면 실행이 영원히 끝나지 않습니다. 페이지를 새로고침하거나 다른 코드 블록을 실행해서 빠져나오세요. while을 쓸 때는 "이 반복은 어떻게 끝나는가?"를 늘 생각하는 습관이 중요합니다.

for vs while — 언제 무엇을?

  • 횟수가 정해져 있다 → for
  • 몇 번 돌지 모른다, 조건으로 멈춘다 → while

예시:

  • "학생 30명 점수 합산" → 횟수 정해짐 → for
  • "정답 맞힐 때까지 묻기" → 횟수 모름 → while
  • "파일 끝까지 읽기" → 끝이 어딘지 미리 모름 → while
이 섹션의 핵심
  • while 조건: — 조건이 True인 동안 반복.
  • 조건을 False로 만들 코드가 안에 반드시 있어야 — 안 그러면 무한 루프.
  • while True: + break — 의도적 무한 + 빠져나오기 패턴.
  • 횟수 정해짐 → for, 조건으로 끝 → while.
SECTION 07

break / continue — 반복 중간에 빠져나오기

반복 중간에 흐름을 조작하는 두 단어입니다.

  • break: 반복 자체를 끝낸다 — 즉시 빠져나옴
  • continue: 이번 반복만 건너뛰고 다음으로

2주차에서 익힌 것
  • input()으로 사용자에게 입력받기 — 결과는 항상 문자열
  • int(), float(), str()로 형 변환 — 새 객체 생성

  • break(반복 종료) / continue(이번만 건너뜀)
  • i 이름표가 매 반복마다 새 객체로 옮겨감 (1주차 모델 일관)

3주차에서는 여러 값을 묶는 리스트, 튜플, 딕셔너리와 직접 만드는 함수(def)를 다룹니다.
